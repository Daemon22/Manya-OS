Grammar: `expr := or := and ('OR' and)* := not ('AND' not)* := 'NOT' not | primary := '(' expr ')' | comparison := operand ('==' | '!=' | 'in') operand`. Operators: `==`, `!=`, `in [...]`, `AND`, `OR`, `NOT`, parentheses. Literals: strings (single/double-quoted with `\` escapes), numbers, `true`, `false`, `null`, lists. - Implemented hierarchy module: `DecisionHierarchy` tree of `DecisionNode`s (id, role, authority, parent). `escalationPath` walks leaf-to-root with cycle detection. `findCommonAuthority` returns the lowest common ancestor by comparing root-first paths. `canOverride` is strict-greater-than authority comparison. `ancestors` returns immediate-parent-first chain (excludes the node itself). `roots`, `children`, `validateHierarchy` (duplicate ids, unknown parents, cycles).
